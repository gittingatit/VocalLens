# VocalLens - Voice Controlled Photo Organizer

VocalLens is like Google photos but smarter- it lets you organize photos using voice commands.

## Setup Instructions
1. Create virtual environment: `python -m venv venv`
2. Activate: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux)
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python main.py`
5. Open: `http://localhost:5000`
